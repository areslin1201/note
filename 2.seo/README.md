## Step1:查詢相關關鍵字
site > 搜尋結果侷限域名，舉例 > site:domain.com

allintitle >　搜尋HTML標記語言<title>中之間的部分，只查詢標題欄，舉例 > site:關鍵字

allinurl > 只查詢網頁連結，舉例 > allinurl:"cgi-bin" phf +com

### Step2:標題設置關鍵字
title規範: 2高1低: 高相關性，高搜尋量，低難度
